# **App Name**: LoanTrack Pro

## Core Features:

- Client Management: Register and manage client information, marking them as new or returning clients.
- Loan Issuance: Record loan details, automatically calculating and applying 20% interest and a 4% upfront processing fee.
- Payment Tracking: Track daily repayments (cash in) and loans given out (cash out) to manage cash flow effectively.
- Client Status Flags: Automatically flag clients who miss payments, exceed repayment days, or consistently repay on time, labeling them as good clients.
- Cash Flow Management: Calculate opening and closing balances automatically, tracking cash in, cash out, and expenses like lunch, transport, and airtime.
- Dashboard Overview: Display total loans given, repayments collected, expenses, and a list of clients overdue or skipping payments.
- Performance tool: Use an LLM as a tool to analyze the performance of each cashier by monitoring their transactions, identifying patterns, and using reasoning to decide if there are deviations from average values.

## Style Guidelines:

- Primary color: Soft blue (#89B6E2) to convey trust and stability.
- Background color: Light gray (#F0F4F8) for a clean and professional look.
- Accent color: Muted teal (#63BCAE) to highlight important actions and information.
- Body text: 'PT Sans', a humanist sans-serif. Headline font: 'PT Sans', a humanist sans-serif.
- Use financial icons such as currency symbols and charts to represent features, reports and categories.
- Design the dashboard to clearly show essential data: loans, payments, cash flow. Implement card-based design for cashier performance.
- Incorporate subtle transitions and animations for changes in values or new data loading to provide smooth UI feedback.
